# Portfolio

Персональный сайт-резюме с информацией об опыте работы, навыках и проектах.

## Технологии

- React 18 + TypeScript
- Vite
- Three.js + React Three Fiber для 3D визуализации
- Framer Motion для анимаций
- Tailwind CSS + SASS
- EmailJS для отправки сообщений
- React Hook Form

## Запуск проекта

### 1. Установка зависимостей

```bash
yarn install
```

### 2. Запуск в режиме разработки

```bash
yarn dev
```

### 3. Сборка для production

```bash
yarn build
```

### 4. Просмотр production сборки

```bash
yarn preview
```

## Структура

Проект содержит следующие основные разделы:

- О себе
- Опыт работы
- Технологический стек
- Проекты
- Контактная форма

Поддерживается переключение между русским и английским языком, а также светлой и темной темой оформления.
